# Rentory

Rentory is a privacy-first rental record organiser for tenants.

## Core features
- Rental records
- Room checklists
- Move-in, during-tenancy and move-out photos
- Document storage
- Timeline events
- Local PDF reports
- Local backup export and import
- Optional App Lock
- One-off lifetime unlock for expanded limits

## Privacy-first principles
- No account needed
- Records stay on your device by default
- No advertising trackers
- No behavioural analytics
- No third-party AI services
- Reports are created locally
- Backups are created locally
- Photos and documents are stored locally

## App boundary
Rentory helps users organise their own rental records. It does not give legal, financial or tenancy advice.
