# Rentory

Rentory is a privacy-first rental record organiser for tenants and landlords. It runs locally on iPhone, iPad, Mac, Apple Vision Pro and Apple Watch.

## Profiles
- Renter — record your own tenancy: rooms, photos, documents, timeline, reminders, exportable PDF.
- Landlord — adds tenancies, rent payments, expenses and compliance reminders (gas safety, EICR, EPC).

## Core features
- Rental records, per-room checklists, move-in / during-tenancy / move-out photos.
- Document storage with type tagging.
- Timeline events for inspections, repairs, deposit discussions, etc.
- Reminders with local 9 AM notifications.
- Local PDF reports with disclaimer.
- Landlord rent + expense tracking with monthly net.
- Local backup export and import (.rentorybackup package).
- Optional iCloud private-database sync for the user's own devices.
- Optional App Lock (Face ID / Touch ID / passcode).

## Glanceable surfaces
- iOS Home Screen widgets (Next reminder, Monthly finance, Next step).
- Apple Watch companion app (Reminders / Records / Quick Add tabs).
- Apple Watch face complications (Next reminder, Record progress) across accessoryCircular, accessoryRectangular, accessoryInline and accessoryCorner families.

## Privacy-first principles
- No account needed.
- Records stay on the device by default.
- No advertising trackers, no behavioural analytics, no third-party AI.
- Reports, backups, photos and documents stay local unless the user exports or shares them.
- The only sanctioned network surfaces are (a) the user's own iCloud private database (opt-in) and (b) StoreKit for the optional lifetime unlock.
- WatchConnectivity moves the snapshot between iPhone and the paired Apple Watch only — nothing leaves the device pair.

## App boundary
Rentory helps users organise their own rental records. It does not give legal, financial or tenancy advice. It is not a property management system.

## One-off lifetime unlock
Adds higher record / room / photo limits, full report export and App Lock. No subscription.
